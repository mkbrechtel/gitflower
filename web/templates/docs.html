Welcome to GitFlower Documentation

GitFlower is a lean, local Git development server providing a read-only web interface for repository visualization and management.

Key Features

  • Repository management with organization folders
  • Read-only web interface for browsing
  • Security through filesystem permissions
  • Local-first, privacy-focused design
  • AI-friendly APIs (MCP support planned)

Quick Start

# Create a new repository
gitflower create myproject.git

# Start the web interface
gitflower web

# Browse to http://localhost:8747
